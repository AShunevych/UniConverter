# UniConverter
App for conversion and calculation .    
Available for conversion: 
- Area   
- Force 
- Time 
- Length 
- Temperature 
- Speed   
- Volume    
- Weight  
     
Other:  
- Calculator
- Online currency converter
     
Minimum Sdk Version - 26 (Android 8.0)       
Locales - English / Ukrainian
